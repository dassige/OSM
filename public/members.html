CSV File Format

Your CSV file must include headers on the first line. The columns can be in any order.

Required: name, email

Handling Commas: If a name contains a comma, wrap it in quotes.

name,email,mobile "Doe, John",john@example.com,021123 Jane Smith,jane@test.com,
